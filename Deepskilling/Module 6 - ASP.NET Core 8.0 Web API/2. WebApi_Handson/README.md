# 2.WebApi_Handson — Web API with Swagger & Postman

## Objectives

- Demonstrate Swagger installation and WebAPI listing in browser
  - Install NuGet: `Swashbuckle.AspNetCore`
  - Use `ProducesResponseType` on action methods
  - `AddSwaggerGen` in services and `UseSwaggerUI` in middleware

- Demonstrate Postman usage to call WebAPI methods
  - Collections, requests, headers (Authorization), JSON body, choosing HTTP verb

- Demonstrate route usage and `Name`/`ActionName`
  - Use user-friendly names for action methods
  - `ActionName` allows multiple methods with same HTTP verb

---

This folder contains a sample project `WebApiWithSwagger` demonstrating these features.
